
# サイト制作仕様まとめ


## 1 制作手順

1. 各ページhtmlを作成。ヘッドの中身調整/ｖｉｅｗ　ｍｏｒｅのボタン
2. 各ページのbodyレイアウト作成。
3. headerのレイアウトを作成。-------------ナビゲーション実装（ハンバーガーメニューの作成・お問い合わせフォームの設置。）
4. footerのレイアウト作成　-------------copyrightの設置。
5. 作成したheader/footer を下層ページへコピー。その際、変更によって生じたヘッドもコピーして変更する。
6. 2個目以降の、製品詳細ページを複製および画像リンクの変更。
7. サイト内すべての動線（リンク）の設置 (index 内の製品リンク、SNSリンク（Instagram/X等）の紐付け。)
8. レスポンシブデザイン実装(モバイルファーストの考え方を採用しデスクトップ用のレイアウト作成)



## 2 条件最終チェック項目

リンク正常性: 全てのリンクが正しく繋がりエラーが出ないか。

パスの整合性: 画像やリンクの指定が相対パスになっており、本番環境で表示されるか。

レスポンシブ: スマホやPCで見た際に崩れがなく、メニューが正常に開閉するか。

JS動作確認: エラーがなく処理が意図通りか.

ご提示いただいた要件を整理し、開発・実装時に確認しやすい形でマークダウンにまとめました。

## 3. 全体デザイン・レスポンシブ仕様

* **共通実装:**
    * **ヘッダー:** ハンバーガーメニュー実装

* **ブレークポイント:** 900px
| ページ種別 | 最大幅 |
| :--- | :--- |

| **About / Company** | 600px |
| **商品詳細ページ** | 800px |

## 4. ページ別実装詳細

### トップページ (index.html)
* **グリッドレイアウト:** PC 4列 / スマホ 2列
* **動線:** 「View More」ボタンで `products.html` へ遷移

### Productsページ (products.html)
* **基本構成:** トップページと同じグリッド仕様
* **機能:** 一覧下部にページネーションを設置し遷移を制御

### 商品詳細ページ (item1.html)
* **構成:** 画像とテキストを横並びに配置
* **動線:** ページ最下部に「Back To Products」リンクを設置

### Companyページ (company.html)
* **コンテンツ:** Google Mapを埋め込み
* **スタイル:** CSSの `filter: grayscale(100%);` を適用してモノクロ化
